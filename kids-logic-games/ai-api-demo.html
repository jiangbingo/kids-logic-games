📚 代码示例

// 配置 API Key aiGenerator.setApiKey('your-api-key-here'); // 设置 API 模式 aiGenerator.setApiMode('auto'); // 'auto', 'mock', 'api' // 生成记忆游戏 const memoryData = await aiGenerator.generateMemoryGame('水果', '4-5岁', '中等'); // 生成形状游戏 const shapeData = await aiGenerator.generateShapeGame('4-5岁', '中等'); // 生成声音游戏 const soundData = await aiGenerator.generateSoundGame('农场', '4-5岁', '中等');

ℹ️ 说明

  • Auto 模式：优先使用 API 生成，失败时自动降级到模拟数据
  • Mock 模式：仅使用模拟数据，不调用 API
  • API 模式：仅使用 API，失败时抛出异常
  • 所有生成的数据都会自动缓存，下次访问时直接使用缓存
  • 如果没有配置 API Key，系统会自动使用 Mock 模式
