# 快速开始 🚀

## 1. 本地运行

```bash
# 启动开发服务器
make start

# 在浏览器中打开
# http://localhost:8000
```

## 2. 运行测试

```bash
make test
```

## 3. 构建生产版本

```bash
make build
```

## 4. 部署到EdgeOne

```bash
make deploy
```

详细的部署说明请参考 [DEPLOYMENT.md](DEPLOYMENT.md)

## 可用命令

```bash
make help    # 查看所有可用命令
make start   # 启动本地服务器
make stop    # 停止本地服务器
make status  # 查看服务状态
make test    # 运行测试
make build   # 构建生产版本
make deploy  # 部署到EdgeOne
make clean   # 清理构建文件
```
