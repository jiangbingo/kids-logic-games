# 记忆大师 🧠

## 游戏玩法

1. 观察3-8张卡片3-6秒
2. 卡片翻转隐藏
3. 根据提示找到目标卡片
4. 正确得分，错误扣分
5. 完成50关

## 关卡设计

- 总关卡：50关
- 4个难度级别

### 难度划分

| 关卡 | 卡片数 | 展示时间 | 分数倍数 |
|------|-------|---------|---------|
| 1-10 | 3张 | 6-4秒 | 1.0x |
| 11-25 | 4张 | 5-3秒 | 1.2x |
| 26-40 | 6张 | 4-2秒 | 1.5x |
| 41-50 | 8张 | 3-1秒 | 2.0x |

## 教育价值

- **记忆力**：提升短期记忆能力
- **观察力**：培养细节观察能力
- **注意力**：锻炼专注能力
- **空间记忆**：提升位置记忆能力

## 游戏流程

1. **准备阶段**（3秒）
   - 倒计时准备

2. **展示阶段**（3-6秒）
   - 显示所有卡片
   - 记住位置

3. **寻找阶段**
   - 卡片翻转隐藏
   - 显示目标提示
   - 点击找到目标

4. **完成阶段**
   - 显示正确答案
   - 进入下一关

## 分数系统

- 每个正确答案：10分 × 难度倍数
- 每个错误答案：-5分 × 难度倍数
- 最低分数：0分（不会出现负分）

## 小技巧

- 快速记住位置
- 注意卡片特征
- 不要着急选择
- 可以重玩关卡
