# 数字认知 🔢

## 游戏玩法

1. 屏幕显示若干个物品（1-5个）
2. 从4个选项中选择正确的数量
3. 正确得分，错误扣分
4. 完成所有5关

## 关卡设计

- 总关卡：5关
- 每关选项：4个
- 得分规则：正确+10分，错误-5分
- 数字范围：1-10

## 物品列表

- 🍎 苹果
- 🌟 星星
- 🐟 鱼
- 🌸 花
- 🎈 气球
- 🦋 蝴蝶

## 教育价值

- **数字认知**：学习1-10数字
- **计数能力**：培养数数能力
- **数学基础**：建立数学思维
- **数量概念**：理解数量的意义

## 操作提示

- 先数一数有多少个
- 选择对应的数字
- 正确获得分数
- 进度自动保存

## 难度

- **适合年龄**：3-6岁
- **难度级别**：简单
- **平均时长**：3-4分钟

## 小技巧

- 一个一个地数
- 用手指指着数
- 数完再选择
- 不要着急
