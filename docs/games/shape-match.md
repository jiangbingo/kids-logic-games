# 形状识别 ⭐

## 游戏玩法

1. 屏幕中央显示一个目标形状
2. 从4个选项中找出相同的形状
3. 正确得分，错误扣分
4. 完成所有5关

## 关卡设计

- 总关卡：5关
- 每关选项：4个
- 得分规则：正确+10分，错误-5分

## 教育价值

- **形状认知**：学习6种基本形状
- **空间感知**：提升图形识别能力
- **几何概念**：培养空间思维
- **观察能力**：锻炼细节注意

## 操作提示

- 点击正确形状获得分数
- 点击错误形状会扣分
- 注意观察形状的特征
- 完成所有关卡庆祝

## 难度

- **适合年龄**：3-6岁
- **难度级别**：简单
- **平均时长**：2-3分钟
