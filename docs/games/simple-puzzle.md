# 简单拼图 🧩

## 游戏玩法

1. 屏幕中央显示一个目标动物
2. 从4个选项中找出相同的动物
3. 正确得分，错误扣分
4. 完成所有4关

## 关卡设计

- 总关卡：4关（4种动物）
- 每关选项：4个
- 得分规则：正确+10分，错误-5分

## 教育价值

- **手眼协调**：培养精细动作能力
- **观察力**：提升细节观察能力
- **配对能力**：锻炼图形识别
- **动物认知**：学习常见动物

## 操作提示

- 点击正确动物获得分数
- 注意观察动物的特征
- 完成所有关卡会有庆祝
- 进度自动保存

## 难度

- **适合年龄**：3-6岁
- **难度级别**：简单
- **平均时长**：2-3分钟
