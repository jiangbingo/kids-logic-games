# 字母配对 🔤

## 游戏玩法

1. 屏幕显示一个大写字母
2. 从2-4个小写字母中找到对应的小写
3. 正确得分，错误扣分
4. 无限关卡递进

## 关卡设计

- 关卡：无限递进
- 每关选项：2-4个
- 难度：自动增加
- 得分规则：正确+10分，错误-5分

## 教育价值

- **字母认知**：学习26个英文字母
- **大小写配对**：掌握大小写关系
- **字母顺序**：熟悉字母表顺序
- **英语基础**：建立英语入门

## 游戏流程

1. 显示目标大写字母
2. 提供2-4个小写字母选项
3. 选择正确的小写字母
4. 正确进入下一关
5. 难度逐渐增加

## 难度递进

- **第1-5关**：2个选项，简单字母（A, B, C...）
- **第6-10关**：3个选项，常见字母
- **第11-20关**：3个选项，相似字母（如 B/D, P/Q）
- **第21关+**：4个选项，复杂字母

## 小技巧

- 先学字母表顺序
- 记住大写和小写的对应
- 注意形状相似的字母
- 多练习提高速度

## 教育延伸

- 学习字母发音
- 学习字母单词
- 学习字母书写
- 建立英语兴趣
