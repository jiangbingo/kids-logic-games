# 颜色配对 🎨

## 游戏玩法

1. 屏幕中央显示一个目标颜色
2. 从4个选项中找出相同的颜色
3. 正确得分，错误扣分
4. 完成所有5关

## 关卡设计

- 总关卡：5关
- 每关选项：4个
- 得分规则：正确+10分，错误-5分

## 教育价值

- **颜色认知**：学习6种基本颜色
- **视觉训练**：提升颜色辨识能力
- **配对能力**：培养分类和比较能力
- **观察力**：锻炼注意力集中

## 操作提示

- 点击正确颜色获得分数
- 点击错误颜色会扣分
- 完成所有关卡会有庆祝动画
- 进度自动保存，下次继续

## 难度

- **适合年龄**：3-6岁
- **难度级别**：简单
- **平均时长**：2-3分钟

## 小技巧

- 先记住目标颜色的特征
- 观察颜色的深浅变化
- 不要着急，慢慢选择
- 答错了也不要担心

## 家长指南

1. 帮助孩子认识颜色名称
2. 引导孩子观察颜色差异
3. 鼓励孩子自己选择
4. 失败时给予鼓励
